# Git 的安装

[[toc]]

## 如何检验安装成功

打开 cmd 或者 PowerShell 或者 bash（Windows 下装 git 附带），输入：

```bash
git --version
```

如果出现类似 `git version 2.39.0` 的输出就（大概）算安装成功。

## Windows 平台

### 通过 Scoop 安装(推荐，但应该需要科学上网)

```powershell
scoop install git
```
